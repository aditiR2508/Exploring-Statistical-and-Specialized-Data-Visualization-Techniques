# Theory:
### Statistical and specialized visualizations are used when basic charts are not sufficient to analyze complex data. These charts help understand distribution, clustering, overlaps, hierarchy, frequency and multidimensional relationships.

## (a) Statistical Visualization Techniques
### 1.	Area Plot
#### ----> Used to show trends over time with filled area below the line.
#### Applications: Sales growth, population growth, cumulative values.
________________________________________
### 2. Pie Chart
#### ---> It is a circular chart used to represent percentage share of categories.
#### Applications: Budget share, market share, department contribution.
________________________________________
### 3. Donut Chart
#### ---> It is a modified pie chart with a center hole used to show composition clearly.
#### Applications: Revenue share, category contribution.

### 4. Boxplot
#### ---> Used to display data distribution using quartiles. It shows:
#### •	Minimum value 
#### •	First Quartile (Q1) 
#### •	Median 
#### •	Third Quartile (Q3) 
#### •	Maximum value 
#### •	Outliers 
#### Applications: Marks analysis, salary distribution, loan amounts.

### 5. Histogram
#### ---> Used to display frequency distribution of numerical data.
#### Applications: Age distribution, marks spread, income ranges.

### 6. Heatmap
#### ---> Uses colors to represent numerical values or correlation strength.
#### Applications: Correlation analysis, attendance patterns, business metrics.
________________________________________
## (b) Specialized Visualization Techniques
### 1. Bubble Plot
#### ---> It is an advanced scatter plot where the bubble size represents a third variable.
#### Applications: Sales vs profit with customer count.

### 2. Treemap
#### ---> It displays hierarchical data using nested rectangles.
#### Applications: Department budget, file sizes, category share.

### 3. Dendrogram
#### ---> It is a tree-like diagram used in hierarchical clustering.
#### Applications: Student grouping, customer segmentation.

### 4. Venn Diagram
#### ---> It shows relationships between sets using overlapping circles.
#### Applications: Subject selection, common interests, set operations.

### 5. 3D Scatter Plot
#### ---> Used to display relationship among three numerical variables.
#### Applications: Income, loan amount, credit score.
